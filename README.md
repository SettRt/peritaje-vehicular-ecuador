# Peritaje Web Simple

Version web estatica del sistema de peritaje vehicular. No requiere npm, Expo, build ni GitHub Actions.

## Archivos

- `index.html`
- `styles.css`
- `app.js`

## Publicar en GitHub Pages

1. Suba estos archivos a la raiz del repositorio.
2. En GitHub vaya a `Settings > Pages`.
3. En `Source`, seleccione `Deploy from a branch`.
4. Seleccione branch `main` y carpeta `/root`.
5. Guarde.

La aplicacion incluye datos demo, historial local, informe imprimible y modelo 3D con Three.js.
